# Clown
Unity(C#)로 1인 개발한 방탈출(escape room) 퍼즐 게임입니다.  
오브젝트를 클릭해 힌트/아이템을 찾고, 3개의 게임 씬을 순차적으로 클리어하면 결과 화면에서 플레이 기록을 확인할 수 있습니다.

개발 형태: 1인 개발(기획~구현~QA 전 과정 단독 진행)  
플레이 방식: 마우스 클릭 기반 상호작용(힌트 탐색 → 진행)  
게임 구성: 메인/튜토리얼 1 / 게임 씬 4 / 결과 화면

## 프로젝트 개요
- 장르: 방탈출 / 퍼즐 / 1회성 플레이(클리어형)
- 플레이 목표: 씬 내 오브젝트 상호작용을 통해 힌트와 아이템을 찾아 진행하고 최종 클리어
- 결과 제공: 플레이 시간 및 획득 아이템 요약

## 1인 개발 범위(내가 한 일)
- 제안서 작성
- 기획서 작성
- 레벨 디자인
- 전체 플로우 차트 및 씬 단위 플로우 차트 설계
- 컨텐츠(퍼즐/힌트/아이템) 구상
- 에셋 수급/적용(외부 에셋 포함)
- 사운드 리소스 탐색/적용
- 구현(C#, Unity)
- 단독 QA(플레이 흐름, 진행 불가/버그 체크)
